Zigzag Iterator

  • Google
  • Leetcode
  • 算法
技术

Problem:

Given two 1d vectors, implement an iterator to return their elements alternately.

分析

因为只有两个List，所以只要来回在两个迭代器之间切换就好了。思路比较简单，但是要考虑各种corner case。

follow-up。如果有多个List，其实只要用一个iterator的List就好了，可以实现在不同iterator之间的切换。
