Paint Fence

发表于 2016-03-18   |   分类于 技术   |   阅读次数

Problem:

There is a fence with n posts, each post can be painted with one of the k colors.

You have to paint all the posts such that no more than two adjacent fence posts have the same color.

Return the total number of ways you can paint the fence.

分析

就是dp解法。维护两个变量；一个代表最后两个是相同颜色的sameColor变量，一个是最后两个颜色不同的diffColor变量。
因此，每当为下一张post图颜色时，有：

  • 和最后一张颜色不一样，那么有(k-1)种可能性；
  • 和最后一张颜色一样，那么就必须保证最后两张颜色不一样；
public class Solution {
    public int numWays(int n, int k) {
        if (n == 0) return 0;
        if (n == 1) return k;
        int sameColor = k;
        int diffColor = k * (k - 1);
        for (int i = 2; i < n; i++) {
        //图相同颜色的可能性和之前的可能性相同，这里暂时存储起来。
            int temp = diffColor;
            //图不同的颜色，则之前最后两张什么情况都没所谓。
            diffColor = (k - 1) * (sameColor + diffColor);
            sameColor = temp;
        }
        return sameColor + diffColor;
    }
}
