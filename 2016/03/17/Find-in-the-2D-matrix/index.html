Search in the 2D matrix

发表于 2016-03-17   |   分类于 技术   |     |   阅读次数

Problem:

Write an efficient algorithm that searches for a value in an m x n matrix. This matrix has the following properties:

  • Integers in each row are sorted in ascending from left to right.
  • Integers in each column are sorted in ascending from top to bottom.

题目

给定一个二位矩阵，这个二位矩阵有如下特性：

  • 每一行从左到右有序且递增；
  • 每一列从上到下有序且递增；

分析：

从最右上角元素开始。会出现以下两种情况：

  • 如果目标元素小于当前元素，那么当前一列元素不可能会出现该元素，因为毕竟每一列是从上到下递增，那么就列index减1；
  • 如果目标元素大于当前元素，那么当前一行不会出现该元素，因为毕竟每一行都是从左到右递增，那么行index加1；
  • 如果相同，返回；

还是比较简单的题目，重点是知道从最右上角作为搜索起点。
