Problem:

Given a set of sorted intervals based on their start time and a new interval; insert this new interval and merge the intervals where necessary.

题目：

给定一系列按照起始点排好序的intervals，然后再给你一个interval，插入到这个已经排好序的intervals中，最后再合并intervals。

Analysis:

有两种思路：

  1. 直接将新的interval插入到intervals中，然后再对intervals排序；
  2. 如果采用上面那种方案，那么题目给定的排好序的intervals就没有利用到。所以我们需要的就是遍历intervals，然后把新的interval插入到合理的位置，使得新的intervals还是按照起始点有序的。
