Problem:

Given a number represented as an array, let one plus this number and return the new array.

题目

给定一个数组表示的数字，数组低位表示数字高位，数组高位表示数字低位。给这个数字加1，然后返回新的数组。

分析

比较简单的题目，值得注意的就是进位情况。

  • 如果数字某一位是9，那么就需要进位；
  • 如果循环结束后，数组最低位为0，证明数字最高位进位，需要开辟新的数组来存储，并且将进位的1放在新的数组头部；
