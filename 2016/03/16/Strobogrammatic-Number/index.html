Strobogrammatic Number 1&2

Leetcode链接/Strobogrammatic Number

Problem 1:

A strobogrammatic number is a number that looks the same when rotated 180 degrees (looked at upside down).

Write a function to determine if a number is strobogrammatic. The number is represented as a string.

For example, the numbers "69", "88", and "818" are all strobogrammatic.

分析：

第一题比较简单，考虑好各种case就能过。

Problem 2:

上题是问你一个字符串是不是Strobogrammatic Number，而这题叫你求出所有的长度为n的Strobogrammatic Number。

分析

肯定是递归，不多说，但是怎么递归。长度为n的所有Strobogrammatic Number和长度
为n－1的Strobogrammatic Number之间有什么关系。嗯！想了半天，没关系，反而长度为n的Strobogrammatic Number和长度为n－2的Strobogrammatic Number很有关系。

来看看为什么n和n-2之间会存在关系，因为Strobogrammatic Number肯定是成对增长的，不可能左边添加了某个数，而在右边没有添加对应的Strobogrammatic Number，这样就不会构成新的Strobogrammatic Number。

  • 长度为0的Strobogrammatic Number是“”；
  • 长度为1的Strobogrammatic Number是0,1,8；
  • 那么长度为2的怎么计算：
    • 将长度为n-2的所有字符串做如下处理：
      • 最左最右加0，0+s+0;
      • 最左最右加1，1+s+1;
      • 最左最右加8，8+s+8;
      • 最左加6，最右加9，6+s+9;
      • 最左加9，最右加6，9+s+6;
    • 但是要特殊处理添加0的情况；
      • 如果不是求解当前长度的Strobogrammatic Number，那么就可以在左右加0，否则左右加0是不合法的数字；

一般在有递归解的方法下，都会想一下是否还存在迭代解法。其实想想也是很简单实现的. 单纯的循环和每次的替换就能够满足题目要求。
